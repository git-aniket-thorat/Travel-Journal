# Travel Journal

A simple and beautiful travel journal application to share your travel experiences and adventures around the world.

## About This Project

This is a travel journal web application that displays your travel memories with photos, locations, and descriptions. Each entry shows where you traveled, when you visited, and memorable details about your trip.

## What This Project Has

### Main Features

- **Travel Header** - Red header with "My Travel Journal" branding at the top
- **Entry Cards** - Each travel entry displays as a card with:
  - Travel photo on the left side
  - Location country with a map marker icon
  - Link to view location on Google Maps
  - Trip title
  - Travel dates (start date - end date)
  - Description of the trip
  - Photos have rounded corners for a modern look

### How It Works

1. **Header Section** - Shows the application title and branding
2. **Entry Display** - Shows each travel journal entry in a card format
3. **Image Display** - Photos appear on the left with fixed size (125px √ó 168px)
4. **Information Section** - All trip details appear on the right side
